### 문제 설명

<p>
N-Queen 문제는 크기가 N × N인 체스판 위에 퀸 N개를 서로 공격할 수 없게 놓는 문제입니다. N이 주어졌을 때, 퀸을 놓는 방법의 수를 구하는 프로그램을 작성하시오.
</p>

### 입력

<p>
첫째 줄에 N이 주어집니다. (1 ≤ N < 15)
</p>

### 출력

<p>
첫째 줄에 퀸 N개를 서로 공격할 수 없게 놓는 경우의 수를 출력합니다.
</p>
