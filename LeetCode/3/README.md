### 문제 설명

<p>
주어진 문자열 `s`에서, 반복되는 문자가 없는 가장 긴 **부분 문자열(substring)**의 길이를 찾는 프로그램을 작성하시오.
</p>

### 입력

<p>
하나의 문자열 `s`가 함수의 인자로 주어집니다.

**예시 1:** `s = "abcabcbb"`
**예시 2:** `s = "bbbbb"`
**예시 3:** `s = "pwwkew"`
</p>

### 출력

<p>
반복되는 문자가 없는 가장 긴 부분 문자열의 길이를 정수형으로 반환합니다.

**예시 1 출력:** `3` ("abc")
**예시 2 출력:** `1` ("b")
**예시 3 출력:** `3` ("wke")
</p>
